# Curriculum Classes and Course Sections

The app uses two different concepts that should stay distinct.

## Curriculum Class

A curriculum class is a math-course/category in the question bank, such as Algebra 2, Pre-Calculus, or AP Calculus.

Curriculum classes organize questions:

- Class
- Unit
- Section

They are about content placement, not rosters.

## Course Section

A course section is an actual group of students in the Gradebook, such as “Period 2 AP Calc”.

Course sections organize grade data:

- Roster
- Enrollments
- Assessment instances
- Scores
- Category weights

Course sections may optionally link to a curriculum class, but they should not be named or modeled as curriculum classes.
